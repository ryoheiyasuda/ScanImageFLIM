Welcome to ScanImageFLIM GitHub Pages.

This software is a variant of ScanImage v.3.8 with FLIM functionality. It is currently compatible with PicoQuant TimeHarp 260 (N or P) and Becker and Hickl 150.

To install, please download ScanImage v 3.8.1 and then add/replace the file structure. In addition, lisenced DLL and headers must be placed in the matlab path.
